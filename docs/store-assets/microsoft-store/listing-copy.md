## Product name

```text
M-Verify
```

## Description

```text
M-Verify helps authorized business staff confirm M-Pesa payments using live payment records. Search by transaction code, amount, or customer, select the matching payment, and verify it without relying on screenshots or verbal claims.

Businesses can send STK Push payment requests, follow payment status, print or save PDF receipts, share receipts through WhatsApp, view transactions, and manage staff access. The compact floating Windows window can stay on top, minimize to the system tray, and start with Windows.

M-Verify works independently from a POS system. It connects securely to your organization's hosted M-Verify account and its configured Safaricom Daraja integration. An account provided by your business administrator and an internet connection are required.
```
